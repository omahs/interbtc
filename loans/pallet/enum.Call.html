add_market

Fields

asset_id: <<T as Config>::Assets as Inspect<<T as Config>::AccountId>>::AssetId
market: Market<<<T as Config>::Assets as Inspect<<T as Config>::AccountId>>::Balance>

Stores a new market and its related currency. Returns Err if a currency is not attached to an existent market.

All provided market states must be Pending, otherwise an error will be returned.

If a currency is already attached to a market, then the market will be replaced by the new provided value.

The lend_token id and asset id are bound, the lend_token id of new provided market cannot be duplicated with the existing one, otherwise it will return InvalidLendTokenId.

  • asset_id: Market related currency
  • market: The market that is going to be stored
